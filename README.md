# 🌎 Verificador de Nacionalidade

Projeto simples desenvolvido em JavaScript para verificar se o usuário é brasileiro ou estrangeiro com base no país digitado.

## Tecnologias utilizadas

- HTML
- CSS
- JavaScript
- Manipulação de DOM

## Funcionalidades

- Campo para digitar o país
- Verificação automática
- Resultado exibido na tela

## Objetivo

Praticar conceitos básicos de JavaScript como:

- Condicionais (if/else)
- Eventos
- Manipulação do DOM
